Tars - App de Aprendizaje de Chino

Aplicación inmersiva para aprender chino que combina flashcards SRS, roleplay con voz impulsado por IA y ejercicios de shadowing con análisis de audio en tiempo real.

Tech Stack: React Native, FastAPI, SQLite, OpenAI (GPT-4o-mini), Groq Whisper, Edge-TTS

Funcionalidades: Aprendizaje de vocabulario con flashcards, roleplay con IA que genera escenarios, análisis de tono y pronunciación, sistema de repetición espaciada.

Heredity - Red Bayesiana

Modelo probabilístico bayesiano para inferir la probabilidad de que cada persona en una familia tenga un rasgo genético particular (ej: discapacidad auditiva causada por mutaciones en el gen GJB2).

Cómo funciona: Realiza enumeración exhaustiva sobre todas las configuraciones posibles de genes/rasgos, calcula la probabilidad conjunta de cada escenario y normaliza los resultados para proporcionar distribuciones interpretables.

Tech: Python, Redes Bayesianas, Modelos Probabilísticos

PageRank - Análisis de Páginas Web

Implementación del algoritmo PageRank para analizar la importancia de páginas web basándose en su estructura de enlaces.

Funcionalidades: Parseo de archivos HTML para construir un grafo de enlaces, cálculo de PageRank mediante muestreo y métodos iterativos, factor de amortiguación configurable.

Tech: Python, Algoritmos, Grafos

Minesweeper - Solver con IA

Solver de Minesweeper que utiliza inferencia lógica y una base de conocimiento de oraciones sobre el estado del juego para deducir celdas seguras y localizar minas.

Cómo funciona: La IA mantiene una base de conocimiento de oraciones lógicas, marca celdas como seguras o minas basándose en pistas, infiere nuevo conocimiento comparando oraciones existentes y elige movimientos seguros cuando es posible.

Tech: Python, Pygame, Inferencia Lógica

CS50AI

Tic Tac Toe - IA Invencible

Juego de Tic Tac Toe con una IA invencible usando el algoritmo Minimax con memorización. La IA siempre juega de manera óptima y nunca pierde.

Cómo funciona: El algoritmo Minimax simula recursivamente todos los movimientos posibles y sus resultados, asigna valores de utilidad a estados terminales (+1 para victoria de X, -1 para victoria de O, 0 para empate) y elige los movimientos que maximizan la probabilidad de ganar.

Tech: Python, Pygame, Minimax, Memoización
